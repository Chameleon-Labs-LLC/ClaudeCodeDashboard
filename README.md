# ClaudeCodeDashboard

Local GUI dashboard for Claude Code — browse sessions, manage memory, search, and inspect usage

## Setup

```bash
npm install
cp .env.example .env.local
# Fill in .env.local with real values
npm run dev
```

## Tech Stack
- Next.js 14, TypeScript, Tailwind CSS
- NextAuth v5
- Prisma + PostgreSQL
- AWS Amplify
